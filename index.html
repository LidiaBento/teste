Principais habilidades

A seguir, estão demonstradas as principais habilidades com as quais estou atuando, sendo elas:

Swift

Swift é uma linguagem de programação consistente e intuitiva, desenvolvida pela Apple para a criação de apps para iOS. Ela foi criada para dar ainda mais liberdade para os desenvolvedores. Swift é para que qualquer pessoa com uma boa ideia consiga fazer coisas surpreendentes.

CoreData

Os CoreData são uma estrutura usada para gerenciar os objetos da camada de modelo em seu aplicativo. Ele fornece soluções generalizadas e automatizadas para tarefas comuns associadas ao ciclo de vida do objeto e gerenciamento de gráfico de objetos, incluindo persistência.

Cocoapod

O Cocoapods é um gerenciador de dependência no nível de aplicativo para o Objective-C, Swift e quaisquer outros idiomas executados no tempo de execução do Objective-C, que fornece um formato padrão para o gerenciamento de bibliotecas externas, utilizados por muitos desenvolvedores.
